# 部署步骤

## Clone the repo

```bash
git clone https://github.com/alexunder/reality-x-ui.git
```

# Run the install script

```bash
cd reality-x-ui/
./install.sh
```

Follow the steps, input the username, password, and the port number. Finally, you complete the setting up.
